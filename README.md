# Environment variables

- NODE_ENV: "production" or "development"
- SERVER_NAME: host name of mongoose db server
- SERVER_PORT: port of mongoose db server

Inspiration:
- https://whitetoothbrewing.com/the-beers-index

Focus:
- performance
- simplicity
- user friendly
- mobile first
